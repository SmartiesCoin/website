What these numbers mean

Regular and Evo masternodes participate equally in every quorum that matters for day-to-day network operation - ChainLocks, InstantSend, and hard-fork signaling. The collateral difference (15k SMT for Regular, 75k SMT for Evo) only changes the masternode role; it doesn't make Evo nodes "count more" in these LLMQs. The single exception is Smartiecoin Platform (LLMQ_100_67), which is Evo-only - and Platform is not a current network priority.

Because Regular nodes cost 5x less collateral, they are the most cost-effective way to grow the network and stabilize the LLMQs that secure transactions and finality today. When Bad Members in a DKG session rise close to the quorum size, the session can fail - this is expected while the network is still growing.
